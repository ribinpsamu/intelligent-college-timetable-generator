# 📖 Project Overview

The Intelligent College Timetable Generator is a web-based timetable management system developed using Flask and MySQL. It automates timetable generation while managing academic resources such as departments, faculty, classrooms, subjects, and student schedules.

The application provides secure role-based access for administrators, HODs, faculty members, and students through an interactive dashboard.
